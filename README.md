# TinPets 🐾

A parody pet dating website — Tinder, but for your furry friends.

**Live site:** https://314dhan.github.io/tinpets/index.html

## About

TinPets is a fun parody project inspired by dating apps like Tinder. Instead of humans swiping for love, it's your pets finding their next best friend. Built as a landing page with a full premium UI redesign featuring animated backgrounds, scroll reveals, and cinematic interactions.

## Pages

- **Home** — hero, features, testimonials, pricing, CTA
- **How It Works** — 3-step walkthrough with interactive mockups
- **Success Stories** — pet match stories and community stats
- **FAQ** — common questions across general, privacy, pricing, and safety

## Tech Stack

- HTML5 / CSS3 (OKLCH colors, scroll-driven animations, clip-path)
- Vanilla JavaScript (IntersectionObserver, Web Animations API)
- Bootstrap 5
- Google Fonts — Playfair Display + Poppins + Fredoka One
